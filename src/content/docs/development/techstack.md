description: Describe the tech stack used in the project
---

The project is based on the `ASP.NET` core hence I have used most of the packages and technologies that blend well with it for easier and faster development.

- **IDE:** Visual studio (2026)
- **Database:** Microsoft SQL
- **Programming Language:** C#
- **Framework:** Asp.net + Blazor
- **SQL management:** SQL server management studio 2022
- **Env management:** DotNetEnv (v3)
- **Auth:** Identity EFC (v9)
- **Report generation:** ClosedXML (v0.105.0)
- **Icons:** Remix icons (via CDN)
- **Logging:** Extending the built-in logger interface with Serilog (v3)
