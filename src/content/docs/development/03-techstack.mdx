## Infrastructure & Packages

### Identity & Security
*   **ASP.NET Core Identity EF Core (v9):** Handles robust, extensible user authentication, authorization, and session management.
*   **DotNetEnv (v3):** Dynamically loads environmental configurations from local `.env` files into the application context.

### Utilities & Logging
*   **ClosedXML (v0.105.0):** A lightweight, strongly-typed .NET companion used for server-side Excel report generation without requiring Interop.
*   **Serilog (v3):** Extends the native ILogger interface to provide robust, structured file and console logging capability.

### UI & Presentation
*   **Visual Studio:** The primary IDE utilized for continuous debugging, package management, and system deployment.
*   **Remix Icons:** Integrated cleanly via a CDN to supply a modern, lightweight vector icon library for the interface.
