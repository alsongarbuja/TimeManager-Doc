The `Button` shared partial view is a reusable UI component designed to standardize button rendering across the application. It provides built-in support for dynamic state management, such as conditional loading spinners, disabled states, and vector icon injection.

## Component Implementation

To ensure strict type safety when rendering the component, the Razor view is bound directly to a dedicated C# View Model (`ButtonViewModel`).

            </svg>

        }
    </button>

## Configuration Parameters

When calling the partial view, you pass a new instance of `ButtonViewModel`. Below are the available properties you can configure:

| Property | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **`Id`** | `string` | `""` | The unique DOM identifier for the button. |
| **`Type`** | `string` | `"button"` | The HTML button type (e.g., `"button"`, `"submit"`, `"reset"`). |
| **`Classes`** | `string` | `""` | Custom CSS utility classes appended to the button. |
| **`Loading`** | `bool` | `false` | If `true`, hides the text/icon and displays a spinning SVG indicator. |
| **`Disabled`** | `bool` | `false` | If `true`, applies the native HTML `disabled` attribute. |
| **`Text`** | `string` | `""` | The primary text label displayed inside the button. |
| **`IconClass`**| `string?`| `""` | Optional Remix Icon class (e.g., `"ri-save-line"`) injected before the text. |

## Example Usage

Here is how the button is utilized inside the authentication portal to submit the login form. Notice how we pass the strongly-typed ViewModel directly into `@await Html.PartialAsync()`.

```razor ins={27-34}
<!-- Views/Auth/Login.cshtml -->
@model TimeManager.Backend.ViewModels.LoginViewModel

@{
    Layout = "_MainLayout";
    ViewData["Title"] = "Login";
}

@section scripts {
    <script src="/js/passwordtoggle.js" defer></script>
}

<div class="login-container">
    <div class="form-wrapper">
        <h2 class="form-title text-center">Welcome back</h2>
        <p class="form-subtitle text-center">Please sign in to your account</p>
    </div>

    <div class="mt-8 form-wrapper">
        <div class="form-card">
            <form class="space-y-6" asp-controller="Auth" asp-action="Login" method="post">
                <div asp-validation-summary="ModelOnly" class="error-alert" role="alert"></div>

                <custom-input asp-for="UserName" />
                <custom-input asp-for="Password" />

                <!-- Button Partial Injection -->
                @await Html.PartialAsync("Button", new TimeManager.Backend.ViewModels.PartialViews.ButtonViewModel
                {
                    Id = "login-submit",
                    Type = "submit",
                    Classes = "btn-login w-100",
                    Text = "Log in"
                })
            </form>
        </div>
    </div>
</div>
```
