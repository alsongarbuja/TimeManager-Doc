Welcome to the **Components Directory**. This section documents the custom, reusable UI elements built using ASP.NET Core Razor Shared Partial Views built **in this project**.

By abstracting complex markup into shared components, we ensure our primary Razor pages remain thin, our styling remains consistent, and centralized logic (such as Role-Based Access Control and dynamic state rendering) is easily maintainable.

---
