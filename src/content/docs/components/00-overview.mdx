Welcome to the **Components Directory**. This section documents the custom, reusable UI elements built using ASP.NET Core Razor Shared Partial Views **in this project**.

By abstracting complex markup into shared components, we ensure our primary Razor pages remain thin, our styling remains consistent, and centralized logic (such as Role-Based Access Control and dynamic state rendering) is easily maintainable.

## Tag Helper Components

Tag Helpers are a powerful feature in ASP.NET Core that allow server-side C# code to participate in creating and rendering HTML elements. By creating custom Tag Helpers, we encapsulate complex Razor logic—such as validation binding, state retention, and dynamic CSS class assignments—into simple, semantic HTML tags, keeping our form views incredibly clean and readable.
